# 📊 Sales Analysis Dashboard

An end-to-end **Data Analytics Project** built using **Excel, SQL, Python, and Power BI** to analyze retail sales data and generate meaningful business insights.

---

# 📌 Project Overview

This project analyzes the Superstore Sales dataset to identify sales trends, customer behavior, regional performance, and product profitability. The project covers the complete data analytics workflow from data cleaning to interactive dashboard creation.

# 💡 Key Business Insights

- West region generated the highest sales.
- Technology category contributed the highest revenue.
- Consumer segment accounted for the largest share of sales.
- Monthly sales showed seasonal trends.
- Higher discounts often resulted in lower profits.
- A small number of customers generated a significant portion of total sales.
